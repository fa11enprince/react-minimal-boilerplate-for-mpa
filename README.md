## React Minimal Boilerplate For MPA
This project is a minimal boilerplate for multi-page application.

### Package Used
- React v16
- Babel Core v6
- Webpack v4

### How to build / run the project

```
$ npm run watch
```
Then on another console.
```
$ npm run start
``` 

### How to add a new page
You will need to add paths in the `entry` property in `webpack.config.js`.
